# Usage

## Commands

### Sync

Run synchronization for all configured calendars:

```bash
syncalpy sync
```

### List

List configured synchronizations:

```bash
syncalpy list
```

### Status

Show last synchronization status:

```bash
syncalpy status
```

## Examples

### Sync two calendars

```bash
syncalpy sync
```

### List configured syncs

```bash
syncalpy list
```
